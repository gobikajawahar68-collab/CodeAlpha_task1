# CodeAlpha Task 1 - Simple E-Commerce Store

## Project Description
This is a Simple E-Commerce Store developed as part of the CodeAlpha Full Stack Development Internship.

## Technologies Used
- HTML
- CSS
- JavaScript
- Django
- SQLite

## Features
- Home Page
- Product Listing
- Product Details
- Shopping Cart
- User Login
- User Registration
- Checkout
- Order Processing

## Project Structure

CodeAlpha_Task1/
│── index.html
│── products.html
│── product_details.html
│── cart.html
│── login.html
│── register.html
│── checkout.html
│── style.css
│── script.js
│── images/

## How to Run
1. Install Python and Django
2. Run:
   python manage.py migrate
3. Run:
   python manage.py runserver
4. Open:
   http://127.0.0.1:8000/
